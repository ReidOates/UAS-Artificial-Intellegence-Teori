# PNR Validator

Passenger Name Record Validation using Deterministic Finite Automata (DFA).

**Academic Project**  
Universitas Bale Bandung

## Overview
A web application capable of validating a simulated Passenger Name Record (PNR) strictly using a Deterministic Finite Automata (DFA) engine. The application utilizes a Modern SaaS Dark Theme and maps visual execution traces of the validation process.

## PNR Format
- Target Format: `LLDDLL` (L = Letter, D = Digit)
- Accepted Language: `[A-Z]{2}[0-9]{2}[A-Z]{2}`
- Alphabet (Σ): `{A-Z, 0-9}`

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript ES6
- No Frameworks, No Backend, No Build Tools.
